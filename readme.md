## 创建型模式 5
### 工厂方法
#### 工厂模式优点

```
1、避免创建者与具体的产品逻辑耦合
2、满足单一职责
3、每一个业务逻辑实现都在所属自己的类中完成
4、满足开闭原则，无需更改使用调用方就可以在程序中引入新的产品类型
```
